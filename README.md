# esp32-led-level-indicator
ESP32 based LED level indicator using an analog potentiometer input

## Hardware
- ESP32
- Potentiometer
- 6x 220 Ohm resistors
- 6x LEDs

## Features
- Smoothed analog input
- LED bar level representation
- Smooth brightness interpolation
- Non-blocking timing using `millis()`

Future improvements:
- Bluetooth volume control
